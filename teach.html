Today in class : Today in class we will be going over strings, integers, floats and IO (input/output) operations.

1) strings

Denoted as string. User input is always taken in as a string.
In Python, strings are characters inside quote signs. You can use double quotes ("string") or single quotes ('string') to denote a string, BUT the quotes must be of the same type. For example, "string' is not valid in Python.

2) integers

Denoted as int. In simpler terms, integers are whole numbers. (ie: ...,-2,-1,0,1,2,3,4...)
Since user input is taken in as a string, we can convert a number taken in as a string to an integer type as follows :

s = '123'
x = int(s) #this is saying int('123')
